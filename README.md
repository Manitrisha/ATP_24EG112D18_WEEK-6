

# ATP_24EG112D18_WEEK-6 🚀

A collection of React-based mini projects and assignments developed during **Week-6** of the training program. This repository demonstrates concepts like **React Components, State Management, Context API, CRUD Operations, User Handling, and Employee Management**.

---

## 📌 Repository Overview

This repository contains multiple React applications:

| Folder Name | Description |
|-------------|-------------|
| `AddingUsers` | Application to add and manage users dynamically |
| `context-app` | Demonstrates React Context API usage |
| `employee` | Employee management application |
| `userList` | Displays and manages user lists |

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/ATP_24EG112D18_WEEK-6.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd ATP_24EG112D18_WEEK-6
```

---

## 3️⃣ Install Dependencies

Move into any project folder and install packages:

```bash
cd AddingUsers
npm install
```

OR

```bash
cd context-app
npm install
```

---

## 4️⃣ Run the Application

```bash
npm start
```

The application will run on:

```bash
http://localhost:3000
```
